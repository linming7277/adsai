## 💡 避免项目暂停的方法

### 方法1: Cloud Scheduler定时访问

```bash
# 创建Cloud Scheduler任务
gcloud scheduler jobs create http keep-supabase-alive \
  --schedule="0 */12 * * *" \
  --uri="https://[your-project-ref].supabase.co/rest/v1/" \
  --http-method=GET \
  --headers="apikey=[your-anon-key]"
```

**说明**: 每12小时访问一次，保持项目活跃

**成本**: $0（Cloud Scheduler免费额度内）

---

### 方法2: 使用Uptime监控服务

使用免费的监控服务（如UptimeRobot）：
1. 添加监控URL: `https://[your-project-ref].supabase.co`
2. 设置检查间隔: 每天
3. 自动保持项目活跃

**成本**: $0
