## 🎯 WCAG 对比度标准

### 标准要求
- **AA 级别（正常文本）**: 对比度 ≥ 4.5:1
- **AA 级别（大文本）**: 对比度 ≥ 3:1
- **AAA 级别（正常文本）**: 对比度 ≥ 7:1
- **AAA 级别（大文本）**: 对比度 ≥ 4.5:1

## 💡 最佳实践

### 颜色使用原则
1. **文本颜色**: 始终确保 ≥ 4.5:1 对比度
2. **大文本**: 可以使用 ≥ 3:1 对比度
3. **交互元素**: 建议 ≥ 3:1 对比度
4. **装饰元素**: 可以更低，但不应传达重要信息

### 深色模式设计
1. **使用更亮的颜色**: 深色背景需要更亮的前景色
2. **降低饱和度**: 避免过于鲜艳的颜色
3. **增加对比度**: 确保元素层次清晰
4. **测试真实设备**: 不同屏幕显示效果不同
