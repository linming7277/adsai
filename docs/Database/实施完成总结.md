## 使用方式

### 日常开发（增量迁移）

```bash
# 1. 创建迁移文件
# services/{service}/migrations/000002_xxx.up.sql

# 2. 推送代码
git push origin main

# 3. 自动执行迁移
# GitHub Actions自动触发
```

### 完全重建（偶尔使用）

```bash
# 1. 创建重置Job（首次）
gcloud run jobs create db-reset-preview \
  --region=asia-northeast1 \
  --image=asia-northeast1-docker.pkg.dev/your-gcp-project-id/adsai-services/db-migrator:latest \
  --set-secrets="DATABASE_URL=DATABASE_URL:latest" \
  --set-cloudsql-instances=your-gcp-project-id:asia-northeast1:adsai \
  --command="/reset.sh"

# 2. 执行重置
gcloud run jobs execute db-reset-preview --region=asia-northeast1 --wait

# 3. 触发迁移
git push origin main
```
