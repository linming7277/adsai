## 技术实现细节

### 自动替换（97% 效率）

使用 Python 脚本批量替换常见模式：

```python
# 模式: METHOD_NOT_ALLOWED
apperr.Write(w, r, http.StatusMethodNotAllowed, "METHOD_NOT_ALLOWED", "Method not allowed", nil)
# 变为:
apiErr := apierrors.New(apierrors.CodeInvalidRequest, "Method not allowed", nil)
apiErr.HTTPStatus = http.StatusMethodNotAllowed
apiErr.WriteJSON(w, r)

# 模式: UNAUTHORIZED
apperr.Write(w, r, http.StatusUnauthorized, "UNAUTHORIZED", "Unauthorized", nil)
# 变为:
apiErr := apierrors.Unauthorized("Unauthorized")
apiErr.WriteJSON(w, r)

# 模式: RATE_LIMIT
apperr.Write(w, r, http.StatusTooManyRequests, "RATE_LIMIT", "rate limited", nil)
# 变为:
apiErr := apierrors.RateLimited(int(rr.RetryAfterMs))
apiErr.WriteJSON(w, r)
```

### 需要手动修复的情况

**13 个实例**需要手动修复，原因是：
1. 函数签名不匹配（Unauthorized/Forbidden 需要参数）
2. 类型转换（int64 → int 用于 RateLimited）
3. 自定义错误详情（OAuth 交换、配额信息、指标错误）
4. 未使用导入清理

### 应用的关键修复

1. **Adscenter oauth.go:99** - OAuth 交换带错误详情
2. **Adscenter bulk.go:214** - 配额超限带计划信息
3. **Adscenter diagnose.go** - 3 个速率限制实例
4. **Adscenter mcc.go** - 2 个 MCC 链接错误
5. **Adscenter abtest.go** - Forbidden 检查 + 指标刷新
6. **Adscenter openapi_impl.go** - 认证 + 速率限制
7. **Adscenter preflight_handler.go** - 速率限制带 retry-after
8. **类型转换** - 4 个文件的 int64 → int（RetryAfterMs）
