# 🎨 AutoAds SaaS 前端 UI/UX 完整设计方案

## 📋 项目理解总结

### 技术栈
- **前端**: Next.js 14 (App Router) + React 18 + TypeScript
- **UI库**: Radix UI (headless components) + Tailwind CSS v3
- **状态管理**: Zustand (全局状态) + TanStack Query (数据获取)
- **认证**: Supabase Auth (Google OAuth)
- **部署**: Google Cloud Run

### 现有架构
- ✅ 扁平化路由结构 (无组织层级)
- ✅ 用户直连模式 (基于 user_id 的数据隔离)
- ✅ 已实现页面: Dashboard, Offers, Tasks, AdsCenter, Settings, Manage
- ✅ 权限系统: RBAC (User/Admin 角色)
- ✅ i18n 支持: react-i18next

---

## 🎯 设计目标

### 核心定位
**AutoAds = AI驱动的广告联盟Offer自动化评估与投放优化平台**

### 用户画像
1. **联盟营销人员** - 需要快速评估和管理大量offers
2. **广告投放专家** - 需要监控广告账号性能和优化ROI
3. **数据分析师** - 需要深度数据洞察和趋势分析

### 设计原则
1. **🚀 效率优先** - 减少操作步骤，提高工作效率
2. **📊 数据可视化** - 清晰展示关键指标和趋势
3. **🤖 AI赋能** - 突出AI功能价值，智能推荐
4. **🎨 现代美学** - 2025年设计风格，避免传统SaaS模板感
5. **📱 响应式设计** - 完美支持桌面和移动端

## 🎨 视觉设计系统

### 设计风格: **Glassmorphism + 渐变色彩**
- **主色调**: 蓝紫渐变 (科技感、AI感)
  - Primary: `hsl(217, 91%, 60%)` → `hsl(260, 91%, 65%)`
  - Accent: `hsl(280, 85%, 70%)` → `hsl(320, 85%, 65%)`
- **辅助色**: 
  - Success: `hsl(142, 76%, 45%)` - 绿色 (评估通过、盈利)
  - Warning: `hsl(38, 92%, 50%)` - 橙色 (需要关注)
  - Error: `hsl(0, 84%, 60%)` - 红色 (风险、失败)
- **中性色**: 现代灰度系统
  - Background: `hsl(220, 14%, 98%)` (浅色模式)
  - Card: 白色 + 毛玻璃效果 (backdrop-blur)

### 排版系统
- **标题字体**: Inter (现代、清晰)
- **正文字体**: Inter (统一性)
- **数据字体**: SF Mono (等宽，适合数字)

### 组件风格
- **卡片**: 毛玻璃效果 + 微妙阴影 + 圆角
- **按钮**: 渐变背景 + 悬停动画
- **表格**: 斑马纹 + 悬停高亮 + 紧凑布局
- **图表**: 渐变填充 + 平滑动画
