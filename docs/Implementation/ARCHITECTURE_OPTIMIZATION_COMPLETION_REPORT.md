### 开发标准
1. **FinalAdapter模式**: 所有数据库服务必须使用 `GetFinalAdapterForService`
2. **三层架构**: 业务操作必须遵循 user → billing → token 数据流
3. **国际化规范**: 用户界面必须使用 `t()` 函数，禁止硬编码
4. **性能监控**: 所有数据库操作自动集成性能监控
