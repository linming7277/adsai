### 检查方法改进

**推荐的完整检查流程**:

```bash
# Level 1: 编译层
npx tsc --noEmit

# Level 2: 代码层
grep -rn "organizationUid\|organizationId:" src/app --include="*.tsx" --include="*.ts"
grep -rn "from.*~/lib/organizations" src --include="*.ts" --include="*.tsx"

# Level 3: 类型层
grep -rn "organization.*:" src/lib --include="*.ts"
grep -rn "interface.*organization" src/lib --include="*.ts" -i

# Level 4: UI 层
grep -rn "Organization" src/app --include="*.tsx" | grep -v "//"

# Level 5: 组件命名
grep -rn "Organization.*:" src/app --include="*.tsx"

# Level 6: 文件级检查
find src -type f \( -name "*.ts" -o -name "*.tsx" \) | xargs grep -l "organization"

# Level 7: 逐文件审核
for file in $(find src -type f -name "*.ts" -o -name "*.tsx" | xargs grep -l "organization"); do
  echo "=== $file ==="
  grep -n "organization" "$file" | head -5
done
```
