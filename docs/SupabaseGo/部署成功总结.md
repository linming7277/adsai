## 🔧 遇到的问题和解决方案

### 问题1: TypeScript编译错误 (3次)

**错误1 - OnboardingPage返回类型**:
```typescript
// ❌ 错误
async function OnboardingPage() {
  redirect(configuration.paths.appHome);
}

// ✅ 修复
async function OnboardingPage(): Promise<null> {
  redirect(configuration.paths.appHome);
}
```

**错误2 - TextField组件使用**:
```typescript
// ❌ 错误
<TextField value={displayName} onChange={...} />

// ✅ 修复
<TextField>
  <TextField.Label>显示名称</TextField.Label>
  <TextField.Input value={displayName} onChange={...} />
</TextField>
```

**错误3 - 事件处理器类型**:
```typescript
// ❌ 错误
onChange={(e) => setDisplayName(e.target.value)}

// ✅ 修复
onChange={(e: React.ChangeEvent<HTMLInputElement>) => setDisplayName(e.target.value)}
```

### 问题2: GitHub Actions认证失败

**错误信息**:
```
ERROR: (gcloud.run.deploy) You do not currently have an active account selected.
```

**根本原因**:
`deploy-cloudrun` 和 `tag-image` jobs使用了不同的认证action，导致认证信息未正确传递。

**解决方案**:
统一使用 `./.github/actions/gcp-auth`，与 `build-image` job保持一致。

```yaml
# Before
- name: Auth to Google Cloud
  uses: google-github-actions/auth@v1
  with:
    credentials_json: ${{ secrets.GCP_SA_KEY }}
    create_credentials_file: false  # ❌

# After
- name: Checkout
  uses: actions/checkout@v4
- name: Authenticate to GCP
  uses: ./.github/actions/gcp-auth  # ✅
  with:
    credentials_json: ${{ secrets.GCP_SA_KEY }}
    project_id: ${{ env.PROJECT_ID }}
```
