### Step 4: 优化统计卡片

**通用统计卡片模板:**

```typescript
function StatCard({
  title,
  value,
  icon: Icon,
  trend,
  badge,
  onClick,
}: {
  title: string;
  value: React.ReactNode;
  icon: React.ComponentType<{ className?: string }>;
  trend?: 'up' | 'down';
  badge?: { label: string; color: string };
  onClick?: () => void;
}) {
  return (
    <Card hoverable={!!onClick} className="cursor-pointer" onClick={onClick}>
      <CardContent className="p-4">
        <div className="flex items-center justify-between">
          <div className="flex items-center gap-3">
            <div className="rounded-lg bg-muted p-2">
              <Icon className="h-5 w-5 text-muted-foreground" />
            </div>
            <div>
              <p className="text-sm text-muted-foreground">{title}</p>
              <p className="text-2xl font-semibold">{value}</p>
            </div>
          </div>

          {badge && (
            <Badge size="small" color={badge.color}>
              {badge.label}
            </Badge>
          )}
        </div>

        {trend && (
          <div className="mt-2 text-xs text-muted-foreground">
            {trend === 'up' ? '↑' : '↓'} 趋势
          </div>
        )}
      </CardContent>
    </Card>
  );
}
```

**使用:**
```typescript
<StatCard
  title="用户总数"
  value={isLoading ? <LoadingSpinner size="sm" /> : userCount}
  icon={UsersIcon}
  trend="up"
  onClick={() => router.push('/users')}
/>
```

## 性能优化建议

### 1. 按需导入

```typescript
// ✅ Good - 只导入需要的
import { Card, CardContent } from '~/components/ui/card';

// ❌ Bad - 导入整个模块
import * as CardComponents from '~/components/ui/card';
```

---

### 2. 避免过度动画

```typescript
// ✅ Good - 只对重要元素使用动画
<FadeInStagger>
  {topItems.slice(0, 10).map(...)}  // 只对前10个使用
</FadeInStagger>

// ❌ Bad - 对大量元素使用动画
<FadeInStagger>
  {items.map(...)}  // 100+个元素
</FadeInStagger>
```

---

### 3. 懒加载 Framer Motion

```typescript
// 如果页面不需要复杂动画，不导入 Framer Motion
// 使用 CSS transition 代替

// ✅ Good - CSS transition
<div className="transition-all hover:-translate-y-1">

// ❌ Bad - 为简单动画导入 Framer Motion
import { motion } from 'framer-motion';
<motion.div whileHover={{ y: -4 }}>
```

## 故障排除

### 问题1: 动画卡顿

**原因:** 元素过多或嵌套过深

**解决:**
```typescript
// 限制动画元素数量
<FadeInStagger>
  {items.slice(0, 20).map(...)}
</FadeInStagger>
```

---

### 问题2: Hover 不生效

**原因:** 未启用 hoverable 或缺少 onClick

**解决:**
```typescript
// 确保同时设置
<Card hoverable onClick={handleClick}>
```

---

### 问题3: TypeScript 错误

**原因:** 组件 props 不匹配

**解决:**
```typescript
// 查看组件定义
// Card 接受 hoverable?: boolean
// HoverCard 接受 onClick?: () => void
```
