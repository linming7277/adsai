   - 识别表现好/差的Offer

2. **上午 9:00-12:00**:
   - 发现新的Offer机会
   - 使用评估功能快速筛选
   - 批量评估多个Offer

3. **下午 13:00-15:00**:
   - 配置补点击任务

   - 查看评估结果

   - 调整广告投放策略

5. **晚上 18:00-19:00**:
   - 查看Ads中心数据更新
   - 总结当日效果
   - 规划明日任务

### 8. 功能协同优化

**协同关系**:

1. **评估 → 补点击**:
   - 评估低分Offer
   - 自动推荐补点击方案
   - 一键创建补点击任务

2. **评估 → Ads中心**:
   - 评估结果指导广告投放
   - 高分Offer优先投放
   - 低分Offer暂停或优化

3. **Ads中心 → 评估**:
   - Ads数据反馈评估模型

   - 持续优化评估算法
