# 订阅系统用户验收测试清单 (UAT Checklist)

## 🎯 测试目标
验证完整的订阅系统功能，包括前端界面、API集成、权限控制和多语言支持。

## 📋 测试前准备

### 环境要求
- [ ] 前端应用已部署并可访问
- [ ] 后端billing服务运行正常
- [ ] gateway-middleware服务已配置
- [ ] 数据库连接正常
- [ ] 测试用户账号已准备（不同订阅层级）

### 测试账号
- [ ] **试用用户**: 未订阅或试用状态
- [ ] **Starter套餐**: 基础订阅用户
- [ ] **Professional套餐**: 专业版用户
- [ ] **Elite套餐**: 高级版用户
- [ ] **管理员**: 拥有管理权限
