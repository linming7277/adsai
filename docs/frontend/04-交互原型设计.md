## 📐 设计原则

### 核心理念

1. **即时反馈** - 每个操作<0.5秒内显示反馈
2. **降低门槛** - 智能默认值，减少用户选择
3. **流程引导** - 清晰的操作路径，自动推进流程
4. **容错设计** - 支持撤销，降低操作风险

### 交互层次

```
L1 - 全局导航     顶部导航栏、侧边栏
L2 - 页面布局     大盘、Offer库、Ads中心、任务中心
L3 - 组件交互     卡片、表格、弹窗、抽屉
L4 - 元素交互     按钮、输入框、下拉菜单
```

## 🎨 动画规范

### 动画时长

| 动画类型 | 时长 | 缓动函数 | 用途 |
|---------|------|----------|------|
| **快速** | 150ms | ease-out | 按钮hover、tooltip |
| **标准** | 300ms | ease-in-out | 弹窗、抽屉、下拉 |
| **慢速** | 500ms | cubic-bezier | 卡片翻转、页面切换 |
| **特殊** | 1000ms+ | 自定义 | 评估动画、数字滚动 |

### 关键动画示例

**1. 卡片翻转（评估动画）**:
```css
@keyframes flip-card {
  0% {
    transform: rotateY(0deg);
    opacity: 1;
  }
  50% {
    transform: rotateY(90deg);
    opacity: 0.5;
  }
  100% {
    transform: rotateY(0deg);
    opacity: 1;
  }
}

.evaluate-card {
  animation: flip-card 0.6s ease-in-out;
}
```

**2. 数字滚动**:
```javascript
function animateNumber(element, target, duration = 1000) {
  const start = 0;
  const startTime = performance.now();

  function update(currentTime) {
    const elapsed = currentTime - startTime;
    const progress = Math.min(elapsed / duration, 1);

    // 使用 easeOutExpo 缓动
    const eased = progress === 1 ? 1 : 1 - Math.pow(2, -10 * progress);
    const current = Math.floor(eased * (target - start) + start);

    element.textContent = current;

    if (progress < 1) {
      requestAnimationFrame(update);
    }
  }

  requestAnimationFrame(update);
}
```

**3. Toast 淡入淡出**:
```css
@keyframes toast-in {
  from {
    transform: translateY(-100%);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}

@keyframes toast-out {
  from {
    transform: translateY(0);
    opacity: 1;
  }
  to {
    transform: translateY(-100%);
    opacity: 0;
  }
}

.toast-enter {
  animation: toast-in 0.3s ease-out;
}

.toast-exit {
  animation: toast-out 0.3s ease-in;
}
```

## 📋 总结

### 交互设计亮点

✅ **智能推荐** - 自动识别、智能推荐，减少用户选择
✅ **即时反馈** - 动画、进度条、Toast，操作可见
✅ **流程引导** - 清晰的操作路径，自动推进流程
✅ **容错设计** - 二次确认、一键撤销，降低风险
✅ **手势友好** - 滑动、长按、下拉，移动端优化

### 核心流程图谱

1. **添加评估** - 粘贴URL → 识别 → 评估 → 结果展示
2. **补点击任务** - 配置参数 → 启动任务 → 进度查询 → 完成
3. **投放到账号** - 选择账号 → 确认投放 → 建立关联 → 产生ROAS
4. **批量操作** - 选择Offer → 预览影响 → 执行 → 结果
5. **一键修复** - 检测风险 → 展示方案 → 确认 → 执行
6. **签到奖励** - 提醒 → 点击签到 → 动画 → 获得Token
