## 📋 优化概述

### 当前路由结构（不合理）

```
/dashboard               → 大盘（Dashboard overview）
/dashboard/adscenter     → Google Ads管理
/dashboard/batchopen     → 批量访问
/dashboard/siterank      → 排名分析
```

**问题分析**:
- ❌ adscenter/batchopen/siterank 都是独立功能模块
- ❌ 不应该作为 dashboard 的子路由
- ❌ URL过长，不利于用户体验
- ❌ 架构不清晰，功能层级关系混乱

### 优化后路由结构（推荐）

```
/dashboard     → 大盘（Dashboard overview）
/adscenter     → Google Ads管理
/batchopen     → 批量访问
/siterank      → 排名分析
/offers        → Offer库（新增）
/tasks         → 任务中心（新增）
```

**优点**:
- ✅ 所有功能模块平级，架构清晰
- ✅ URL简洁，用户体验好
- ✅ 符合功能模块的独立性
- ✅ 易于扩展新功能

### Step 2: 创建OAuth回调页面

**文件**: `apps/frontend/src/pages/adscenter/callback.tsx`

```typescript
import { GetServerSidePropsContext } from 'next';
import { useRouter } from 'next/router';
import { useEffect } from 'react';
import { withAppProps } from '~/lib/props/with-app-props';
import RouteShell from '~/components/RouteShell';
import { useApiRequest } from '~/lib/hooks/use-api-request';

const AdsCenter CallbackPage = () => {
  const router = useRouter();
  const { code, state } = router.query;
  const { execute: handleOAuthCallback, loading } = useApiRequest();

  useEffect(() => {
    async function processCallback() {
      if (!code || !state) return;

      // 验证 state
      const savedState = sessionStorage.getItem('oauth_state');
      if (state !== savedState) {
        console.error('OAuth state mismatch');
        router.push('/adscenter?error=invalid_state');
        return;
      }

      try {
        // 调用后端处理OAuth回调
        const result = await handleOAuthCallback({
          path: '/adscenter/oauth/callback',
          method: 'POST',
          body: { code, state },
        });

        // 清除state
        sessionStorage.removeItem('oauth_state');

        // 重定向到adscenter并显示成功消息
        router.push('/adscenter?success=true');
      } catch (error) {
        console.error('OAuth callback error:', error);
        router.push('/adscenter?error=oauth_failed');
      }
    }

    processCallback();
  }, [code, state]);

  return (
    <RouteShell
      title="Google Ads OAuth"
      description="Processing OAuth callback..."
    >
      <div className="flex items-center justify-center min-h-screen">
        <div className="text-center">
          <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-gray-900 mx-auto mb-4"></div>
          <p className="text-gray-600">正在连接Google Ads账号...</p>
        </div>
      </div>
    </RouteShell>
  );
};

export default AdsCenterCallbackPage;

export async function getServerSideProps(ctx: GetServerSidePropsContext) {
  return await withAppProps(ctx);
}
```

## 🔄 兼容性考虑

### 旧URL重定向（可选）

如果需要保持旧URL的兼容性，可以添加重定向规则：

**方法1: Next.js配置重定向**

**文件**: `next.config.js`

```javascript
module.exports = {
  async redirects() {
    return [
      {
        source: '/dashboard/adscenter',
        destination: '/adscenter',
        permanent: true, // 301永久重定向
      },
      {
        source: '/dashboard/batchopen',
        destination: '/batchopen',
        permanent: true,
      },
      {
        source: '/dashboard/siterank',
        destination: '/siterank',
        permanent: true,
      },
    ];
  },
};
```

**方法2: 中间件重定向**

**文件**: `apps/frontend/src/middleware.ts`

```typescript
import { NextRequest, NextResponse } from 'next/server';

export function middleware(request: NextRequest) {
  const pathname = request.nextUrl.pathname;

  // 重定向旧路由
  const redirects = {
    '/dashboard/adscenter': '/adscenter',
    '/dashboard/batchopen': '/batchopen',
    '/dashboard/siterank': '/siterank',
  };

  if (pathname in redirects) {
    return NextResponse.redirect(
      new URL(redirects[pathname], request.url),
      { status: 301 }
    );
  }

  return NextResponse.next();
}

export const config = {
  matcher: '/dashboard/:path*',
};
```

## 🎯 总结

### 核心改变

1. ✅ **路由扁平化**: 从2层嵌套改为1层平级
